# Magi UI Synthwave | Zed Theme

This package contains the Zed implementation of the Magi UI Synthwave theme.

## Files

- `themes/magi-ui-synthwave-zed.json` — Zed theme JSON (`v0.2.0` schema)

## Install (local)

Copy the theme file into your Zed themes directory:

```bash
mkdir -p ~/.config/zed/themes
cp platforms/zed/themes/magi-ui-synthwave-zed.json ~/.config/zed/themes/
```

Then open Zed and select:

- **Magi UI Synthwave Dark**

## Notes

- The palette is ported from the JetBrains Magi UI Synthwave colors for visual consistency.
- This target is file-based only (no build step required).
