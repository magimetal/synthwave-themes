# Magi UI Synthwave | Slack Theme

This package contains the Slack implementation guidance for the Magi UI Synthwave theme.

## Files

- `themes/magi-ui-synthwave-slack.txt` — Slack sidebar import strings (current 4-color + legacy 8-color compatibility)

## Install

In Slack, open **Preferences → Themes**, then paste one of the import strings from:

- `platforms/slack/themes/magi-ui-synthwave-slack.txt`

## Notes

- Recommended current import string (4-color): `#13051f, #200933, #d21e85, #9963ff`
- Legacy compatibility import string (8-color): `#13051f,#200933,#d21e85,#ffffff,#541f73,#ffffff,#55c6e1,#9963ff`
- This target is plain text only (no build toolchain required).
